Current Focus

At the moment, I am working as a research assistant on an HCI research project that involves working with light and colour sciences. We are looking to formulate a model that can allow us to determine a change in perceptible colour based on the luminance at a given point in our physical world. This is a very interesting project as it involves a lot of math and physics, which makes the computer science part of the project far more interesting for me. See the :projects page for more information.

In my spare time when I'm not enjoying the summer break, I am working on a side project that I call Nano. Nano is a C WASM module that is designed to allow a developer to attach compute shaders directly to the webpage using WebGPU compute shaders. So far, for debugging purposes I worked on getting ImGui working with WebGPU + C and I got it to render out properly onto the swapchain. Using Nano, I would like to try and implement a simple 2D fluid simulation for the web before the next term starts.
